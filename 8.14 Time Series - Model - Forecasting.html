Stationarity Check¶

We will write a function for performing rolling statistics and dickey fuller test.

In [8]:
def test_stationarity(timeseries):
    # determining rolling statistics
    rolmean = timeseries.rolling(window=52, center=False).mean()
    rolstd = timeseries.rolling(window=52, center=False).std()
    
    # plotting rolling statistics
    orig = plt.plot(timeseries, color='blue', label='Original')
    mean = plt.plot(rolmean, color='red', label='Rolling Mean')
    std = plt.plot(rolstd, color='black', label='Rolling Std')
    plt.legend(loc='best')
    plt.title('Rolling Mean & Standard Deviation')
    plt.show(block=False)
    
    # let's perform the dickey fuller test
    print('Results of Dickey-Fuller Test')
    dftest = adfuller(timeseries, autolag='AIC')  # initialize
    dfoutput = pd.Series(dftest[0:4], index=['Test Statistic', 'p-value', '#Lags Used', 'Number of Observations Used'])
    # initializing the critical value variable
    for key, value in dftest[4].items():
        dfoutput['Critical Value (%s)'%key] = value
    print(dfoutput)

Now that we have made the time series stationary, let's make models on the time series. We will use statistical modelling method called ARIMA to forecast the data where there are dependencies in the values.

To find out the p and q values for the model, we will use the ACF and PACF plots.

Plotting ACF¶

plt.axhline(y=7.96/np.sqrt(len(ts_log_mv_diff)), linestyle='--', color='gray')
plt.title('Autocorrelation Function')
plt.show()

The AFC curve crosses the upper confidence value when the lag value is between 0 and 1. Thus the optimal value of q in the ARIMA model must be 0 or 1.

The PACF drops to 0 between lag values 1 and 2. Thus the optimal value of p in the ARIMA model is 1 or 2.

When using ARIMA, we do not explicitly differentiate because ARIMA takes care of that so we can use the ts_log (decomposed data) and leave the differencing to be done by ARIMA
