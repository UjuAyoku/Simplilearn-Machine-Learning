Hypothesis testing and p-value¶

Using the null hypothesis, let's assume that 'There is no relationship between TV ads and sales'.

Let's test this hypothesis. We shall reject the null hypothesis if the 95% confidence interval does not include o i.e.

Reject null hypothesis if p<0.05 Fail to reject null hypothesis if p>0.05

In [14]:
import statsmodels.formula.api as smf  # import statsmodel
lm = smf.ols(formula='Sales ~ TV', data=data).fit()   # apply ols on tv sales
In [15]:
# confidence interval of the fitted parameters
lm.conf_int()

The output above shows the p values for the model coefficients. The p value represents the possibility that the coefficient is actually 0.

In this case, the p-value for TV is far less than 0.05 hence we infer that there is a relationship between TV ads and sales.

Feature Selection¶

How do we know what features need to be included in a linear model?

Tips

  1. Try out different models and only keep the predictions/features if they have small p-values

  2. Check if the rquared value goes up when you add new predictors.

Note: rsquared will always increase when you add features to the model even if they are unrelated to the response. So selecting a model with the highest rsquared is not a reliable approach for choosing the best linear model.
