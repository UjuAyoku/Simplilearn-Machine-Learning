Let's see which of the columns are required for our analysis¶

Notice that Cabin has almost 80% missing values. We will drop it completely.

What about Age? Age seems like a relevant predictor for survival. We want to keep it but it has 177 missing values. We need to find a way to approximate those missing values and drop variables that aren't valuable for predicting survival.

We should at least keep the following variables:

  • Survived
  • Pclass: Does the class influence survival rate
  • Sex
  • Age: Does age impact the survival rate?
  • SibSp: Does the no of relatives on the boat that are siblings or spouse affect a persons survivability?
  • Parch: Does the no. of relatives on the boat that are children or parents affect a persons survivability?
  • Fare: Does the fare a person pay affect his survivability?
  • Embarked: Does a person's point of embarking matter? It depends on how the boat was filled; let's keep it.

A person's name, ticket number and passenger ID are irrelevant to predicting his survivability. As you recall, the Cabin variable is mostly missing values so let's drop all of these:

From the above analysis, we could say that the younger a passenger is, the more likely it is for them to be in third class. The older a passenger is, the more likely it is for them to be in first class.

We need to deal with the missing values in the Age variable. Let's write a function to find each null value in the age variable:

In [13]:
def age_approx(cols):
    Age = cols[0]
    Pclass = cols[1]
    
    if pd.isnull(Age):
        if Pclass == 1:
            return 37
        elif Pclass == 2:
            return 29
        else:
            return 24
    else:
        return Age

Let's interprete this result:

Out of the 166 actual instances of NOT SURVIVED, the classifier predicted correctly 143 of them.

Out of the 101 instanced of SURVIVED, the classifier predicted correctly 79 of them.

The first row is about the not-survived-predictions: 143 passengers were correctly classified as not survived (called true negatives) and 23 where wrongly classified as not survived (false positives).

The second row is about the survived-predictions: 22 passengers where wrongly classified as survived (false negatives) and 79 where correctly classified as survived (true positives).
